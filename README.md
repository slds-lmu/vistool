# vistool

<!-- badges: start -->

[![r-cmd-check](https://github.com/slds-lmu/vistool/actions/workflows/r-cmd-check.yaml/badge.svg)](https://github.com/slds-lmu/vistool/actions/workflows/r-cmd-check.yaml)
<!-- badges: end -->

The goal of `vistool` is to visualize optimization traces and aid in
teaching optimization-related concepts.

`vistool` helps you: - explore loss landscapes and objective functions
in one or two dimensions; - inspect optimization runs for gradient
descent variants; - communicate model behavior with publication-ready
plots.

## Get started

Installation steps, optional Python requirements, and hands-on examples
now live in the “Getting started” vignette: - Local:
`vignette("vistool", package = "vistool")` - Online:
<https://slds-lmu.github.io/vistool/articles/vistool.html>

The vignette walks through package setup, plotly surface rendering, and
a complete mlr3-based workflow.
