# Conduct time decay to adjust the update. See https://neptune.ai/blog/how-to-choose-a-learning-rate-scheduler

Conduct time decay to adjust the update. See
https://neptune.ai/blog/how-to-choose-a-learning-rate-scheduler

## Usage

``` r
step_size_control_decay_time(decay = 0.01)
```

## Arguments

- decay:

  (`numeric(1)`)  
  The decay parameter indicating how fast the step size is reduced.

## Value

The step size as number.
